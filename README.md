# MATH QUIZ GAME

# Game Overview
A browser-based math quiz game where players solve arithmetic problems across three difficulty levels. The game features a timed challenge with 10 problems per round, score tracking, hint system, and visual feedback.

Game Objective & Rules
Objective
Score maximum points by correctly solving arithmetic problems within the time limit.

# Rules Game Flow:

10 math problems per game session

60-second time limit (extendable with correct answers)

Three difficulty levels: Easy, Medium, Hard

# Scoring System:

Correct answer: +10 points

Wrong answer: -3 points

Using hint: -5 points

Time bonus: +5 seconds for correct answers

# Problem Types by Level:

Easy: Addition/subtraction (1-10)

Medium: Addition/subtraction/multiplication (1-50)

Hard: All operations including division (1-100)

# Special Features:

Wrong answers become disabled (cannot be re-selected)

Hint removes two incorrect options

Auto-solve reveals correct answer

Progress bar shows completion status

Confetti animation on game completion
